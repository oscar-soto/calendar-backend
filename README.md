# Calendar Backend

This is a MERN Example

main dependencies:
- bcryptjs
- cors
- express
- jsonwebtoken
- moment
- mongoose

**NOTE:**

You need to have installed `nodemon` globally
`npm install nodemon -g
`
## Instalation
- Clone this repository
- Use the next command:
`npm install`
- Add file .env for required environment variables (example: .env.example)

## Use

##### Use local
`npm run dev`

##### Use Production
`npm run start`
